# Beginia

这个项目提供了一个随身的面板，包含 todos，随笔，灵感等的记录。

这个项目将提供各种前端的接入，包括网页、手机 App、甚至单片机，以支持真正的随身携带。

Beginia Android: https://github.com/kyliancc/kyc-beginia-android

## 技术栈

### 后端

- Golang
- SQLite
- Gin
- Docker

### 前端（计划中）

#### Web

- JavaScript
- Vue.js / React.js

#### Android

- Kotlin
- Jetpack Compose

#### MCU

- ESP32
